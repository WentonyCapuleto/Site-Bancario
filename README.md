# 🏦 Clone de Interface Bancária (Projeto Acadêmico)

Projeto desenvolvido exclusivamente para fins acadêmicos com o objetivo de reproduzir a interface de uma aplicação bancária, praticando conceitos de desenvolvimento Front-end, responsividade, validação de formulários e experiência do usuário (UX).

> **Aviso:** Este projeto **não possui qualquer vínculo com o Itaú ou qualquer outra instituição financeira**. Nenhum dado informado é armazenado, transmitido ou utilizado para autenticação. Utilize apenas informações fictícias durante os testes.

---

# 📖 Sobre o Projeto

O principal objetivo deste projeto é aplicar conhecimentos em desenvolvimento web por meio da reprodução fiel de uma interface bancária, simulando o fluxo de preenchimento de formulários e navegação para fins de aprendizado.

Durante o desenvolvimento foram explorados conceitos de estruturação de layouts, estilização, validação de campos e manipulação do DOM.

---

# 🚀 Funcionalidades

- Interface inspirada em um ambiente bancário
- Layout responsivo
- Formulários com validação de campos
- Máscaras para CPF e outros campos
- Manipulação do DOM com JavaScript
- Feedback visual para o usuário
- Simulação de fluxo de navegação
- Organização modular dos arquivos

# 🔒 Segurança

Este projeto possui finalidade exclusivamente educacional.

- Não realiza autenticação.
- Não envia dados para servidores.
- Não armazena informações dos usuários.
- Não utiliza APIs bancárias.
- Não coleta credenciais reais.

Todos os dados utilizados durante os testes devem ser fictícios.
